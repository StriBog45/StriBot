Список команд бота: https://vk.cc/a6Giqf

Для работы требуется авторизовать канал, кнопка: "Авторизовать канал"
Если хотите использовать другое имя для бота, требуется авторизовать бота на аккаунте с желаемым именем. Кнопка: "Авторизовать бота"

Настройки повторяемых сообщений находятся в файле appsettings.json , открыть можно как тестовый файл

В UserSettings.bin сохраняются ваши авторизационные данные.

В каталоге "Отчеты" создается список команд при закрытии программы
В каталоге "Currencies" сохраняются варианты названий валюты
В каталоге "Database" находится база данных, в которой хранится накопленная валюта зрителями
